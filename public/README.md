# Public Assets

This folder contains static assets served at the site root.

## Required Assets

The following files should be added before final public launch:

| File | Purpose | Recommended specs |
|------|---------|-------------------|
| `favicon.ico` | Browser tab and bookmark icon | 32×32 or 16×16, ICO format |
| `og-image.png` | Open Graph image for social sharing | 1200×630 px, PNG format |

**Replace these with official Northbridge Venture Group branded assets before launch.** Placeholder or default assets may currently be in use.
