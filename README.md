# Northbridge Venture Group

A corporate website for Northbridge Venture Group, a holding company that owns and develops projects including Aviator Network, AirTax Financial, and Royal Flight School.

## Tech Stack

- **Next.js 14** (App Router)
- **TypeScript**
- **Tailwind CSS**

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
