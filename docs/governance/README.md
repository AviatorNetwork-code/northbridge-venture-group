# Northbridge Governance

Corporate governance documents for Northbridge Venture Group. These records define **what is officially approved, complete, or frozen**—separate from operating standards (NBS), knowledge (NKS), and application code.

---

## Read order

1. **Governance registry** — `GOVERNANCE-REGISTRY.md` (index of all governance documents)
2. **Applicable governance doc** — for the domain in question
3. **NBS / NKS / code** — implementation must align with approved governance

When governance conflicts with ad-hoc requests, **governance wins** until formally amended.

---

## Proposing changes

1. Draft an amendment or new NB-GOV document.
2. Register it in `GOVERNANCE-REGISTRY.md`.
3. Implement only after status is **Approved**.

Structural website changes after NB-GOV-001 require governance approval. See [NB-GOV-001 — Not Allowed Without Governance Approval](./NB-GOV-001-MARKETING-WEBSITE-V1-COMPLETE.md#not-allowed-without-governance-approval).
