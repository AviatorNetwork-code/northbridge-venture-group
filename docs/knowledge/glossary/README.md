# Glossary

**Purpose:** Shared terms across NBS, NKS, and future CAT. Disambiguate internal vs client-facing language.

---

## Core terms

| Term | Definition |
|------|------------|
| **NBS** | Northbridge Business Standards — how Northbridge works (`docs/business/`, `AGENTS.md`) |
| **NKS** | Northbridge Knowledge System — what Northbridge knows (`docs/knowledge/`) |
| **CAT** | Contextual Advisory Tool (future) — how Northbridge reasons with NBS + NKS |
| **Assessment** | Public `/digital/assessment` funnel; deterministic scoring in code |
| **Recommended solution** | Assessment output label; maps to NKS `solutions/` |
| **Constraint** | Diagnostic gap (internal); maps to assessment pains/systems |
| **Lead category** | Internal qualification label (Hot, Qualified, Nurture, Low Fit) — not client-facing |
| **Evidence** | Rule-level scoring factors from assessment engine — internal only |
| **Ventures** | Northbridge-owned products (not client services) |
| **Services** | Northbridge Digital client offerings |
